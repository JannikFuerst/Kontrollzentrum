# Kontrollzentrum — Open Beta (v1.1.0)

Kontrollzentrum ist dein persönliches App-Dashboard: Web- und Desktop-Apps an einem Ort, mit Favoriten, Kategorien und schneller Suche.

## Features
- Apps hinzufügen (Web / Desktop / Scan)
- Kategorien frei anlegen und verwalten
- Favoriten markieren
- Schnellstart per Klick

## Open Beta
Diese Version ist die erste öffentliche Beta (v1.1.0).  
Feedback ist willkommen – Bugs, Ideen und UX-Verbesserungen gerne melden.

## Entwicklung
```
npm run tauri dev
```

## Build
```
npm run tauri build
```
